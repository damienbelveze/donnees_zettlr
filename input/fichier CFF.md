fichier permettant de lire et retrouver facilement les noms des contributeurs du logiciel ainsi que les licences attribuées à ce logiciel (nécessaires dans le cas d'un hébergement dans [[Zenodo]] et [[Software Heritage]])

syntaxe utilisée dans le fichier CFF : https://citation-file-format.github.io/
Pour créer un tel fichier : https://citation-file-format.github.io/cff-initializer-javascript/#/start


> The concept of a CITATION file has first been used by the R-project [43] and has also been adapted by GNU Octave [9]. This file contains information on how to cite the associated software project in other works. Besides a sample citation, a suggested BibTeX code is often provided in this file 

(source: [[@fehrBestPracticesReplicability2016]])
